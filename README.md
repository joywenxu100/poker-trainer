# 德州扑克决策流程工具 - 使用说明

## 📱 工具介绍

这是一套帮助你在打德州扑克时保持思维严谨、避免低级错误的辅助工具。

包含两个版本：
1. **交互式网页版** (`poker_checklist.html`) - 可以勾选、计时、统计
2. **图片版** (`poker_checklist_image.html`) - 精美流程图，适合截图保存

## 📋 7阶段决策流程详解

### 阶段1: 信息收集 (3-5秒)
- 我的位置是什么？(BTN/CO/MP/EP/SB/BB)
- 我的有效筹码是多少BB？
- 当前底池大小是多少？
- 对手是谁？(紧/松/激进/被动)

### 阶段2: 牌力评估 (2-3秒)
- 我的绝对牌力如何？
- 相对于对手范围的强度？
- 我有多少改进潜力(outs)？
- 公共牌面结构？

### 阶段3: 对手范围分析 (3-5秒)
- 对手之前的行动表示什么范围？
- 对手可能的强牌/中等牌/诈唬比例？
- 对手会不会在这里诈唬？
- 对手会不会fold掉更好的牌？

### 阶段4: 赔率计算 (2-3秒)
- 底池赔率是多少？(pot odds)
- 我需要多少胜率才能call？
- 我的实际胜率/equity是多少？
- 隐含赔率(implied odds)是否足够？

### 阶段5: 情绪自检 ⚠️ (1-2秒) **必检！**
- 我现在是否冷静？
- 刚才的结果是否影响我？
- 我是基于逻辑还是情绪做决策？
- 如果是1小时前，我会做同样决策吗？

### 阶段6: 行动决策 (2-3秒)
- Fold: 我的胜率不足以支撑call/raise
- Call: 我有足够赔率，但raise价值不大
- Raise: 我要value bet/保护底池/诈唬
- Sizing: 我的bet/raise大小合理吗？(通常0.5-0.75pot)

### 阶段7: 二次确认 (1-2秒)
- 这个决策符合我的整体策略吗？
- 这个决策长期来看是+EV的吗？
- 我有没有漏掉关键信息？

**总用时: 15-25秒**

## 💡 使用技巧

### 初期（第1-2周）
- **严格执行**：每个决策点都看一遍流程
- **不要急**：宁可慢一点，也要完成所有步骤
- **重点关注**：阶段5（情绪自检）最容易被忽略

### 中期（第3-4周）
- **逐渐内化**：部分简单决策可以快速过一遍
- **关键时刻**：大池、关键决策必须完整流程
- **记录错误**：每次违反流程的后果

### 后期（第5周+）
- **肌肉记忆**：流程已经变成本能
- **持续提醒**：图片仍然放在旁边，防止松懈
- **复盘优化**：根据实战调整流程细节

## 🌐 在线部署方案（可选）

如果你想让第二个手机直接通过网址访问，而不用传输文件：

### 方法1: GitHub Pages（免费，推荐）

1. **注册GitHub账号**（如果没有）

2. **创建新仓库**
   - 仓库名：`poker-checklist`
   - 设置为Public

3. **上传文件**
   - 上传 `poker_checklist_image.html`
   - 重命名为 `index.html`

4. **启用GitHub Pages**
   - 进入仓库设置 → Pages
   - Source选择 `main` 分支
   - 保存后会生成网址：`https://你的用户名.github.io/poker-checklist/`

5. **手机访问**
   - 第二个手机浏览器输入这个网址
   - 添加到主屏幕，像APP一样使用

### 方法2: 使用免费网盘分享

1. 上传到百度网盘/阿里云盘等
2. 生成分享链接
3. 第二个手机通过链接下载并用浏览器打开

### 方法3: 本地WiFi传输（无需网络）

1. 两个手机连接同一WiFi
2. 使用"文件传输助手"类APP
3. 直接传输HTML文件到第二个手机

## 🎯 培养肌肉记忆的方法

### 1. 离线训练
- 不在牌桌上时，反复看流程图
- 模拟场景，在脑中走一遍流程
- 每天看10-15分钟，持续2周

### 2. 实战应用
- 前100手：严格按流程，不着急
- 第100-500手：逐渐加快速度
- 第500手+：已经形成习惯

### 3. 错误复盘
- 每次输钱后，问自己：
  - 我走完流程了吗？
  - 哪个阶段跳过了？
  - 情绪检查做了吗？

### 4. 量化追踪
- 记录每天完成流程的手数
- 记录违反流程的次数
- 每周复盘改进
