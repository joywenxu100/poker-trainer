# 如何生成JPG图片

## 方法1：双击运行（最简单）

1. 找到文件：`poker_image_generator.py`
2. **双击这个文件**
3. 如果弹出选择程序，选择"Python"
4. 等待几秒，图片会自动生成到**桌面**
5. 桌面会出现文件：`poker_checklist.jpg`

---

## 方法2：命令行运行

打开命令提示符（CMD），输入：

```
python poker_image_generator.py
```

---

## 方法3：直接使用网页版截图

1. 打开文件：`poker_checklist_image.html`
2. 手机截图保存即可

---

## 生成的图片：

- **文件名**: poker_checklist.jpg
- **保存位置**: 桌面
- **尺寸**: 1080 x 3200 像素
- **格式**: JPG (高质量，95%)

---

## 使用方法：

1. 把桌面上的 `poker_checklist.jpg` 发送到手机
2. 打牌时全屏查看
3. 每个决策点扫一眼，确保不遗漏

---

## 如果遇到错误：

**错误1: 找不到PIL模块**
```
pip install pillow
```

**错误2: 找不到字体**
不影响使用，会使用默认字体

**错误3: 无法保存到桌面**
检查桌面路径是否存在

---

- 打印出来贴在桌边
- 发送到多个设备
