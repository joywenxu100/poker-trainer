# 德州扑克训练师 - 使用说明 📱

## 🎯 简介

这是一个专门为德州扑克玩家设计的**手机训练APP**，帮助你：
- ✅ 记住7步标准化决策流程
- ✅ 形成肌肉记忆，避免低级错误
- ✅ 追踪训练进度，解锁成就
- ✅ 随时随地训练，无需联网

**特点**：
- 📲 PWA应用，无需下载安装
- 🔌 支持离线使用
- 💾 数据保存在手机本地
- 🚀 打开即用，无需审核

---

## 📱 如何安装到手机？

### **方法1：iPhone (iOS / Safari)**

1. **用Safari浏览器打开** `poker_trainer.html`
   - 可以通过微信/QQ发送文件到手机
   - 或者上传到网盘/GitHub后访问链接

2. **添加到主屏幕**：
   - 点击底部「分享」按钮 (方框+向上箭头)
   - 滑动找到「添加到主屏幕」
   - 点击「添加」

3. **完成！**
   - 桌面会出现「德州扑克训练师」图标
   - 点击图标，像原生APP一样使用

### **方法2：Android (Chrome / Edge)**

1. **用Chrome或Edge浏览器打开** `poker_trainer.html`

2. **添加到主屏幕**：
   - 点击右上角菜单 (三个点)
   - 选择「添加到主屏幕」或「安装应用」
   - 点击「添加」

3. **完成！**
   - 桌面会出现APP图标
   - 点击使用，支持离线

## 🎓 训练流程详解

### **完整训练模式示例**：

1. **点击「完整训练」**
   - 开始计时

2. **查看问题**
   - 显示当前阶段和问题
   - 例如："阶段1: 信息收集"
   - 问题："我的位置是什么？"

3. **点击卡片翻转**
   - 查看答案和详细解释
   - 理解为什么要这样思考

4. **自评记忆**
   - ✅ **记住了**：该阶段熟练度+2%
   - ❌ **不太记得**：该阶段熟练度-1%

5. **继续下一题**
   - 完成所有26个问题

6. **完成庆祝**
   - 显示训练次数、准确率
   - 检查是否解锁成就

## 💡 使用建议

### **训练计划**

**第1-2周：建立基础**
- 每天3-5次完整训练
- 不要求快，确保每个问题都理解
- 重点记忆7个阶段的名称和顺序

**第3-4周：强化记忆**
- 每天5-10次训练
- 混合使用完整训练和快速复习
- 尝试随机测试，检验效果

**第5-8周：形成习惯**
- 每天至少5次训练
- 重点使用随机测试模式
- 追求更高的准确率

**第9周+：持续巩固**
- 每天2-3次快速复习
- 实战前用随机测试热身
- 维持肌肉记忆

### **最佳实践**

1. **固定时间训练**
   - 早上起床后
   - 午休时间
   - 睡前10分钟

2. **利用碎片时间**
   - 等车时
   - 排队时
   - 休息间隙

3. **实战结合**
   - 打牌前：快速复习一遍
   - 打牌中：脑中过一遍流程
   - 打牌后：针对薄弱环节训练

4. **重点强化阶段5**
   - ⚠️ 情绪自检最容易忽略
   - 每次训练都要认真对待
   - 这是避免Tilt的关键

## 🎮 训练技巧

### **记忆技巧**

**1. 口诀记忆**：
```
信息收集看位置
牌力评估要客观
对手范围多思考
赔率计算要精准
情绪自检最关键 ⚠️
行动决策有理由
二次确认防遗漏
```

**2. 场景联想**：
- 想象真实打牌场景
- 每个问题对应一个具体画面
- 视觉化记忆更牢固

**3. 分组记忆**：
- 前置阶段（1-4）：信息+分析
- 核心阶段（5）：情绪自检 ⚠️
- 决策阶段（6-7）：行动+确认

### **提升熟练度**

**针对薄弱阶段**：
1. 查看统计页面
2. 找到分数最低的阶段
3. 重点训练该阶段的问题
4. 理解每个问题的深层含义

**追求速度**：
- 目标：15-25秒完成一个决策
- 计时器超过30秒会变红
- 在保证准确的前提下提速

## 🔧 技术说明

### **PWA特性**

✅ **离线可用**
- 首次打开后，所有文件缓存到手机
- 无网络也能正常训练
- 数据保存在浏览器本地存储

✅ **像原生APP**
- 添加到主屏幕后独立运行
- 全屏显示，无浏览器地址栏
- 加载速度快

✅ **数据安全**
- 所有数据仅保存在本地
- 不上传任何信息
- 更换设备需要重新训练

### **数据存储**

- 使用 `localStorage` 保存数据
- 占用空间 < 1MB
- 清除浏览器数据会丢失训练记录
- 建议定期截图保存进度

### **兼容性**

✅ 支持：
- iOS Safari 11.1+
- Android Chrome 45+
- Android Edge
- Android Firefox

❌ 不支持：
- 微信/QQ内置浏览器（功能受限）
- IE浏览器

## 🐛 常见问题

### **Q1: 添加到主屏幕后打不开？**
A: 
- 确保4个文件在同一目录
- iOS用Safari浏览器添加
- Android用Chrome/Edge添加

### **Q2: 数据丢失了怎么办？**
A:
- 检查是否清除了浏览器数据
- 更换设备数据不同步
- 建议定期截图保存进度

### **Q3: 能否导出数据？**
A:
- 当前版本不支持导出
- 可以截图保存统计页面
- 或者记录总训练次数

### **Q4: 离线真的能用吗？**
A:
- 首次需要联网打开一次
- 之后完全离线可用
- 飞行模式也能训练

### **Q5: 多个设备能同步吗？**
A:
- 当前版本不支持同步
- 每个设备独立记录
- 建议主要在一个设备上训练
