# 使用指南 📱

## 快速开始

### 第一步：获取API密钥

你需要至少一个AI模型的API密钥：

#### 1. Claude (Anthropic) 🤖
- 访问：https://console.anthropic.com/
- 注册账号并创建API密钥
- 密钥格式：`sk-ant-api03-...`
- 特点：最强大的推理能力，优秀的图片理解

#### 2. Google Gemini 🌟
- 访问：https://makersuite.google.com/app/apikey
- 使用Google账号登录
- 创建API密钥
- 密钥格式：`AIza...`
- 特点：快速响应，免费额度大

#### 3. DeepSeek 🚀
- 访问：https://platform.deepseek.com/
- 注册账号
- 在API密钥页面创建
- 密钥格式：`sk-...`
- 特点：强大的中文能力，性价比高

### 第二步：配置应用

1. 用浏览器打开 `index.html`
2. 点击右下角的 ⚙️ 按钮
3. 输入你的API密钥
4. 点击"保存设置"

💡 **提示**：API密钥只保存在你的浏览器本地，绝对安全！

### 第三步：开始使用

#### 文字问答

1. 在输入框输入你的问题
2. 点击"询问所有模型"
3. 等待几秒钟
4. 查看三个AI的回答并对比

**示例问题**：
- "请解释一下量子计算的基本原理"
- "帮我写一个Python快速排序算法"
- "翻译这段话成英文：..."
- "推荐一些高效学习的方法"

#### 图片分析

1. 点击"选择图片"按钮
2. 选择要分析的图片
3. （可选）输入具体问题
4. 点击"询问所有模型"

**适用场景**：
- 📄 识别图片中的文字（OCR）
- 🎨 描述图片内容
- 📊 分析图表数据
- 🔍 识别物体和场景
- 💡 解答图片中的问题

**示例用法**：
- 上传截图 + 问："这个错误怎么解决？"
- 上传菜单 + 问："帮我翻译这个菜单"
- 上传图表 + 问："分析这个数据趋势"
- 上传手写笔记 + 问："识别并整理这些笔记"

## 常见问题

### ❓ 为什么有些模型没有回答？

可能原因：
1. **未配置API密钥**：检查设置中是否已保存该模型的密钥
2. **API密钥无效**：确认密钥是否正确、是否过期
3. **余额不足**：检查账户是否有足够余额
4. **网络问题**：确认能访问对应的API服务
5. **请求超时**：网络较慢时可能超时，可重试

### ❓ 图片无法识别？

检查以下几点：
- Claude和Gemini支持图片，DeepSeek主要支持文字
- 图片格式：支持 JPG、PNG、GIF、WebP
- 图片大小：建议 < 5MB
- 图片清晰度：确保文字清晰可读

### ❓ 响应速度慢？

正常情况：
- Gemini：通常 2-5 秒
- Claude：通常 3-8 秒
- DeepSeek：通常 2-6 秒

如果明显更慢：
1. 检查网络连接
2. 图片可能过大，尝试压缩
3. 问题过于复杂，AI需要更多思考时间
4. API服务可能繁忙，稍后重试

### ❓ API密钥安全吗？

**非常安全！**
- ✅ 只存储在浏览器本地（localStorage）
- ✅ 不会上传到任何服务器
- ✅ 不会被其他网站访问
- ✅ 代码开源可审查

但请注意：
- ⚠️ 不要在公共电脑上保存密钥
- ⚠️ 定期更换API密钥
- ⚠️ 不要分享你的密钥给他人

### ❓ 会产生费用吗？

**是的**，使用AI API通常会产生费用：

- **Gemini**：有免费额度，超出后按使用量计费
- **Claude**：按Token计费，约 $3-15/100万Token
- **DeepSeek**：较便宜，约 $0.14-0.28/100万Token

**省钱技巧**：
1. 精简问题，避免重复提问
2. 优先使用有免费额度的Gemini
3. 定期检查API使用量
4. 只配置需要的模型

### ❓ 可以离线使用吗？

**不可以**，因为：
- 需要调用在线AI服务
- 需要网络连接发送请求
- AI模型运行在云端服务器

但可以：
- 保存历史对话（手动复制）
- 将HTML保存到本地使用
- 在任何有网络的设备上使用
